Contexto#

Este proyecto conecta mi formación en matemáticas aplicadas con la ingeniería de datos para resolver un problema concreto de previsión y detección de anomalías.

Enfoque#

  • Análisis exploratorio y preparación de datos

  • Visualización de predicciones y anomalías detectadas

Competencias demostradas#

  • Machine learning: modelos supervisados y no supervisados
  • Modelización matemática: enfoque riguroso basado en mi formación académica
