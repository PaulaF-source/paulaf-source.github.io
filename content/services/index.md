## 🔧 Data Engineering sur GCP

**Pour les entreprises qui ont des données dispersées et veulent les centraliser.**

Vos données sont dans des systèmes différents (ERP, fichiers, APIs, IoT) et sont difficiles à exploiter ensemble. Je vous aide à construire une infrastructure de données fiable et automatisée.

**Ce que je fais concrètement :**
- Centralisation de vos sources de données sur Google Cloud Platform
- Création de pipelines ETL automatisés (ingestion, transformation, chargement)
- Administration et optimisation de votre data lake sur BigQuery
- Orchestration avec Apache Airflow

**Stack** : GCP (BigQuery, Cloud Storage, Dataflow, Cloud Run), Apache Airflow, Docker, Python, SQL

## 📊 Analytics & Business Intelligence

**Pour les entreprises qui veulent des tableaux de bord pour prendre de meilleures décisions.**

Vous avez des données mais pas de visibilité claire sur vos KPIs. Je crée des dashboards qui transforment vos données brutes en informations actionnables.

**Ce que je fais concrètement :**
- Création de dashboards Power BI connectés à vos sources de données
- Modélisation SQL pour calculer vos indicateurs clés
- Analyse de performance : logistique, commercial, marketing, opérationnel
- Formation de vos équipes à l'utilisation des dashboards

**Stack** : Power BI, SQL (BigQuery, SQL Server), Python (Pandas), Looker Studio

---

## 📈 Analyse Prédictive & Modélisation

**Pour les entreprises qui veulent anticiper plutôt que réagir.**

Vous voulez aller au-delà du reporting : prévoir la demande, détecter des anomalies, optimiser vos ressources. Mon background en mathématiques me permet d'apporter des solutions analytiques avancées.

**Ce que je fais concrètement :**
- Modèles de prévision (demande, ventes, flux)
- Détection d'anomalies dans vos données opérationnelles
- Optimisation mathématique (positionnement, allocation, tournées)
- Analyse statistique avancée

**Stack** : Python (Scikit-learn, Pandas), SQL, modélisation mathématique
